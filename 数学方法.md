# 四舍五入

```java
        double a=1.2;
        System.out.println(Math.round(a));
```

#  随机数

```java
        //得到一个0-1之间的随机浮点数（取不到1）
        System.out.println(Math.random());
```

```java
         //得到一个0-10之间的随机整数 （取不到10）
        System.out.println((int)( Math.random()*10));//强制转化
```
